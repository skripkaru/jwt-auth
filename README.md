# JWT Auth

JWT Authentication and Authorization

## Tech Stack

**Client:** React, TypeScript, Redux-Toolkit, RTK Query, REST API

**Server:** JSON Server

## Features

- JWT Authentication
- JWT Authorization
- CRUD

## Installation

Install with yarn

```bash
  yarn install
```

Compiles and hot-reloads for development

```bash
  yarn start
```

Compiles and minifies for production

```bash
  yarn build
```

Lints and fixes files

```bash
  yarn lint
```

Format files

```bash
  yarn format
```
